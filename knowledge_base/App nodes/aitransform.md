## Properties & Settings
These are the settings you can configure for this node:

- **instructions** (button): Provide instructions on how you want to transform the data, then click 'Generate code'. Use dot notation to refer to nested fields (e.g. address.street).
- **codeGeneratedForPrompt** (hidden): Code Generated For Prompt
- **jsCode** (string): Generated JavaScript
