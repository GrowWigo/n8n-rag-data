# Node: Read PDF

**Name**: readPDF
**Category**: input
**Description**: Reads a PDF and extracts its content
**Version**: 1

## Properties & Settings
These are the settings you can configure for this node:

- **binaryPropertyName** (string): Name of the binary property from which to read the PDF file
- **encrypted** (boolean): Encrypted
- **password** (string): Password to decrypt the PDF file with
